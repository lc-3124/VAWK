## 基于文字终端的界面也能拥让人惊叹的表现力
### 作为一个tui狂热爱好者，我打算写一个方便开发tui软件的类库

# VAWK ( lc's Visual Ansi-base Widget Kit )
- [ ] 一个用于方便快捷构建tui程序的cpp类库
- [x] 基于文字终端，支持各种绘制和输出的工具包
- [ ] 在终端实现的键盘鼠标检测和独立的消息系统
- [ ] 简单的控件、窗口、绘图设备关系 
- [ ] 易于上手的简单语法和详细的中英文手册
- [ ] 跨平台性好，可以在不同的机器上与操作系统上编译(主要依赖于VA-untils的一些特性，具体请移步此仓库`lc-3124/VA-untils`)

# 关于VAWK
`VAWK` 这个项目我在初二时就有计划，但当时的编程水平一般，加上家长对此的误解，一直没有足够的开发时间和可以正常工作的机器。
在于是我租了一台debian服务器，用屏幕键盘和PUTTY在学校教室编写代码。
在半年前我得到了名为`蓝牙无线键盘`的宝贝，于是利用班上电教委员的职务之便来写一下小项目。
基本上开发环境都是`ssh`+`tmux`+`bash`+`nvim`
也正是因为如此我才自称狂热爱好TUI。

VAWK 被创建的原因另一是一些其他的TUI库不太符号我的口味，所以我只是想自己写一个玩一玩，

实际上VAWK将实现的功能我早已自己实验过，从绘制规则到点击检测和控件系统等等已经在我的机器上跑起来过了。
但是还不满足一个库该有的稳定性，无论是性能上还是封装性上。都还需要大量的调试与修改。

# 关于用户手册
尽管VAWK这个项目还没有真正的开发到可以用的状态，我也决定先开始编写用户手册。
原因就是我的身份问题，作为一个高二的学生，从明年暑假开始，我就必须暂时停止一切与高中课程无关的任何活动
换而言之，到2026年以前，我只有几个月的时间进行开发
